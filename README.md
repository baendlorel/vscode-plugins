# VS Code Plugins Monorepo

> A pnpm-based monorepo for developing and publishing VS Code extensions.

## 📦 Extensions

This monorepo contains the following VS Code extensions:

| Extension                                         | Description                                                                              |
| ------------------------------------------------- | ---------------------------------------------------------------------------------------- |
| [colorful-markdown](plugins/colorful-markdown/)   | Color markdown elements with configurable background, color, decoration and font weight. |
| [colorful-titlebar](plugins/colorful-titlebar/)   | Adds backgroundColor (generated by your project name) to the title bar.                  |
| [jetbrains-titlebar](plugins/jetbrains-titlebar/) | JetBrains-style titlebar with project-based backgroundColor.                             |
| [memory-monitor](plugins/memory-monitor/)         | Monitor memory usage of specified processes in real-time.                                |
| [secret-notes](plugins/secret-notes/)             | Encrypt and decrypt text files in VS Code with AES-256-GCM.                              |
| [simple-launcher](plugins/simple-launcher/)       | Quickly launch commands while monitoring program memory usage.                           |
| [clautcher](plugins/clautcher/)                    | A new VS Code extension (work in progress).                                              |

## 🏗️ Workspace Structure

```
vscode-plugins/
├── plugins/           # Extension packages
│   ├── _shared/       # Shared utilities and types
│   ├── colorful-markdown/
│   ├── colorful-titlebar/
│   ├── jetbrains-titlebar/
│   ├── memory-monitor/
│   ├── secret-notes/
│   └── simple-launcher/
├── configs/           # Shared configuration files
├── scripts/           # Build and utility scripts
└── .vscode/           # VS Code launch configurations
```

**Workspace Configuration:**
- **Package Manager:** pnpm
- **Workspace Config:** pnpm-workspace.yaml
- **Package Location:** `plugins/*`

## 🚀 Getting Started

### Prerequisites

- Node.js (>= 18)
- pnpm (>= 8)
- VS Code

### Installation

```bash
# Install dependencies
pnpm install
```

### Development

```bash
# Build all extensions
pnpm build

# Watch mode for TypeScript compilation
pnpm watch

# Type check without emitting
pnpm check

# Run tests
pnpm test

# Run linter
pnpm lint
```

### Debugging Extensions

Each extension has a dedicated launch configuration in [`.vscode/launch.json`](.vscode/launch.json):

1. Open the extension you want to debug in VS Code
2. Press `F5` or select **Run and Debug**
3. Choose the extension from the dropdown:
   - `启动: simple-launcher`
   - `启动: colorful-markdown`
   - `启动: colorful-titlebar`
   - `启动: jetbrains-titlebar`
   - `启动: memory-monitor`
   - `启动: secret-notes`

## 📜 Available Scripts

| Command           | Description                              |
| ----------------- | ---------------------------------------- |
| `pnpm install`    | Install all dependencies                 |
| `pnpm build`      | Build all extensions                     |
| `pnpm test-build` | Build extensions for testing             |
| `pnpm watch`      | Watch mode for TypeScript compilation    |
| `pnpm check`      | Type check without emitting files        |
| `pnpm test`       | Run all tests                            |
| `pnpm lint`       | Run linter (oxlint)                      |
| `pnpm manifest`   | Generate manifests                       |
| `pnpm pub`        | Build and publish to VS Code Marketplace |
| `pnpm lines`      | Count lines of code                      |

## 🛠️ Tech Stack

- **Build Tool:** Rollup + TypeScript
- **Testing:** Vitest
- **Linting:** oxlint + Prettier
- **Package Manager:** pnpm with workspace support
